# system 檔案介紹
![](https://i.imgur.com/sB3XO1C.png)
## Compiler 參數
### 輸入參數
    - question “題號”
    - test ”測試檔案“
### 內部參數
    - in_path “正確輸入 檔案路徑”
    - ans_path “正確輸出 檔案路徑”
    - out_path “測試輸出 檔案路徑”
    - err_path “測試錯誤資訊 檔案路徑”
### 回傳
    - state "題目狀態"
    - ans "正確答案"
    - out "測試輸出"
    - err "錯誤資訊"
